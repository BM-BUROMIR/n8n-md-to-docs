# n8n Markdown to Google Docs Converter

A service that converts Markdown documents to Google Docs with support for LaTeX formulas, tables, and rich formatting.

## Features

- ✅ LaTeX formulas (inline and display)
- ✅ Tables with proper formatting
- ✅ Numbered and bulleted lists
- ✅ Text formatting (bold, italic, code)
- ✅ Headings, blockquotes, horizontal rules
- ✅ Docker deployment ready

## Quick Start

```bash
# Build and run with Docker
docker compose up -d

# Or run locally
npm install
npm run build:server
npm start
```

See [DEPLOY.md](DEPLOY.md) for detailed deployment instructions.
